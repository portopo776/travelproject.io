# travelproject
Учебный проект, создание простого сайта.

Вояж-Тур — учебный лендинг туристической компании.

Простой адаптивный сайт туристического агентства, созданный в рамках учебного проекта.
Реализовано переключение между страницами (Главная, О компании, Контакты) без перезагрузки, слайдер изображений, форма обратной связи и карта.

🔧 Технологии

    HTML5

    CSS3 (Flexbox, Grid, CSS-переменные, медиазапросы)

    JavaScript (навигация, слайдер, мобильное меню)

    Font Awesome (иконки)

    Google Fonts (шрифты Montserrat, Open Sans)

📁 Структура проекта

├── index.html          # главный HTML-файл

├── styles.css          # все стили проекта

├── img/                # изображения (фоны, карточки туров)

└── README.md           # описание проекта

✨ Особенности

    Адаптивный дизайн (корректно отображается на мобильных устройствах)

    Плавный переход между страницами

    Автоматический слайдер с паузой при наведении

    Бургер-меню на экранах меньше 768px

    Интерактивная Яндекс.Карта на странице контактов

    Форма консультации с базовой валидацией

 🚀 Запуск

Просто откройте index.html в любом современном браузере.
Для корректной работы необходимы изображения в папке img/.
